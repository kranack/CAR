*** 1/ Introduction

Ce logiciel utilise le framework Spree pour Ruby on Rails.
Le site Amazone gère les stocks des livres qu'il propose sur son site e-commerce. 
Site web possèdant deux interfaces :
	- frontend => permet d'effectuer des achats de livres.
	- backend => permet à un administrateur de gérer le site.


*** 2/ Notes

	- Initialiser spree : spree install
	- Lancer le serveur : rails server

*** 3/ Architecture

	L'architecture du site est propre au framework Spree. Les pages du site web ont été générées
	automatiquement par le framework et sont présentes dans les dossier /var/lib/spree-*
